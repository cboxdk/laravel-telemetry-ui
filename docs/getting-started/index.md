# Getting started

The fastest path from `composer require` to a working dashboard: install the
package, point it at the Tempo / Loki / Prometheus you already run, and open the
UI. No agent, no vendor cloud, no data copied out of your infrastructure.

- [Installation](installation.md) — require the package, register connections,
  gate the route, open the dashboard.

Once it's up, keep going with [Core concepts](../core-concepts/) to understand
connections, scope and how the screens are built.
